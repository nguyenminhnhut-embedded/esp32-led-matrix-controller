# ESP32 LED Matrix Web

ESP32 điều khiển LED ma trận 32x8 (MAX7219),
nhập chữ qua webserver AP, tốc độ chỉnh bằng biến trở.

## Hardware
- ESP32
- LED Matrix 32x8 (MAX7219)
- Biến trở 10kΩ

## Wiring
![wiring](Schematic_LED_Matrix_ESP32.png)

## Usage
- Kết nối WiFi: ESP32-MATRIX
- Mở trình duyệt: 192.168.4.1
- Nhập chữ để hiển thị
